Introduction

The jqxScheduler widget displays a set of Appointments in Day, Week, Month, Timeline Day, Timeline Week and Timeline Month views.

Features

  • Multiple Views - Day, Week, Month, Timeline Day, Timeline Week and Timeline Month
  • Edit Dialogs
  • Recurring Appointments
  • Context Menus
  • Keyboard navigation
  • Cells Selection
  • Easy Navigation through Views
  • Customizable Appearance
  • Binding to JSON, XML, TSV, CSV or Array through jqxDataAdapter
  • Resources
  • Dynamic Apointments customization
  • Work Time configuration
  • Time zones
  • Time Scales configuration
  • Cross-browser/cross-platform support
